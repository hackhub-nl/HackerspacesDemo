# hackerspaces-api-demo  

### Instructions:
- Install all dependencies:    
```
npm install
```            
    
- Create file:
```
.env
```   
in the root of the project.   
    
- Set values required in 
```
config\default.ts
``` 
and 
```
config\test.ts
```           
   
- Start the server: 
```
npm run dev
```
