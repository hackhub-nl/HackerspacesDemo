# hackhub-api-poc  

### Requirements
PostgreSQL installed.    

### Instructions:
Create database in PostgreSQL.     
     
Install dependencies:    
```
npm install
```            
      
Replace ```.env.example``` with ```.env``` and set values.         
        
Start the server: 
```
npm run dev
```          
          
Run all tests:    
```
npm test
```
